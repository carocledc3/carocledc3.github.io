Gallery of Works

Java Applications

In my time as a student of the University of the Cordilleras, I have made a few Java console applications as homework. These are the ones I'm particularly proud of. These ones are well-commented, too (at least for my professor's standards).

  • 30CF:02Q2 - Base N to Base N Converter

    Can convert numbers of any base to any base, from bases 2 to 16 inclusive.

  • 30IP:05L2 - Cities Weather Tracker

    Simple application that allows entering of floating-point daily temperature data for an arbitrary number of cities, made possible by regex-powered comma separation.

  • 30IP:06L4 - Music Playlist Manager

    A sample console application that uses doubly-circular linked lists to manage a sample playlist — to add, delete, and select songs within the playlist. Music not included, sadly.

Web Experiments

I had been asked to make a few sample web-development projects in my time there. For all these projects, I can claim credit to most — if not all — of the source code.

  • 10WD-Q2PT Devs Variety

    Our group submission for our Web Development class, Grade 11, 2nd quarter. A sample website that introduces our group, as short-lived as said semester was. I really should've used Tailwind for this one.

  • Squad Tritium

    Our group submission for our jQuery/Web Development class, Grade 12, 1st quarter. Another sample introduction page, this time for a group of three.

  • Romeo's Webdev Snippets'

    My solo submission for our jQuery/Web Development class, Grade 12, 2nd quarter. Meant to be a portal to all the activities I had submitted to that class.

  • (WIP) Pen-Name Personal Webpage

    Okay, so this one is a bit of an exception — this isn't under any of my STI Baguio web projects, nor is this one isn't under my real name at all (thought that might change in the future as I get older), but under my internet pen-name Jonathan. I plan this page to be less professional and more geared towards my hobbies rather than a full portfolio page. But if you do go there, don't tell Jonathan that I sent you.
